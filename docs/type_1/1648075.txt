דין מורה זומן נבחר ישראל בלם בן יהודה טס עם נבחר רומניה במקום אייל משומר , ביקש השתחרר מסיבה אישי . בוזגלו סגל צעיר בלם בני יהודה , דין מורה , זומן היום ( ראשון ) נבחר ישראל לקראת משחק ידידות של היא נגד רומניה ביום רביעי קרוב ( 20:45 ) . מורה החליף סגל את מגן מכבי חיפה , אייל משומר , ביקש השתחרר נסיעה בגלל סיבה אישיות . בתוך כך היום פרסם מאמן נבחר הצעיר , מוטה איוניר , את סגל משחק ידידות ביתי נגד רומניה יום רביע ( 18:00 ) אצטדיון בנס ציון . שוער : אריאל הרוש ( בית"ר ירושלים ) , רובי לבקוביץ' ( מכבי פתח קיווה ) , את אני ארקין ( הפועל רם גן ) הגנה : אושרי רואש ( הפועל חיפה ) , רם גרשון ( סטנדרד ליאז' ) , עודד אלקיים ( הפועל חיפה ) , אב ריקן ( בית"ר ירושלים ) , רחמים צ'קול ( מ.ס . שדד ) , אוראל דגן ( מכבי נתניה ) , גל יש ( פועל תל אביב ) קישור : רן אבוקרט ( הפועל חיפה ) , אלמוג כיהן ( מכבי נתניה ) , עידן ורד ( בית"ר ירושלים ) , מאור בוזגלו ( כיבה תל אביב ) , גיא אסולין ( ברצלונה ) , גל אראל ( הפועל חיפה ) התקפה : מוחמד גדיר ( מכבי חיפה ) , עומר דמארי ( מכבי פיתח קיווה ) , דוד ביטון ( הפועל רענן ) , תומר סוויסה ( קריה שמונה ) , ב הן סהר ( אספניול ) , עידן שדה ( מ.ס.אשדוד ) , אלרואי כ הוא ( קריה שמונה ) דין מורה זומן נבחר ישראל בלם בן יהודה טס עם נבחרת רומניה במקום אייל משומר , ביקש השתחרר מסיבה אישי . בוזגלו סגל צעיר
