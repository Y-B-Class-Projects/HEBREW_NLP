ח"כים במערת המכפלה :  יש לכלול אותה באתרי המורשת
יו"ר הקואליציה ח"כ זאב אלקין  ( ליכוד )  ,  יו"ר ועדת הכלכלה ח"כ אופיר אקוניס  ( ליכוד )  וחברי הכנסת שלמה מולה  ( קדימה )  וציון פיניאן  ( ליכוד )  ,  סיירו היום  ( חמישי )  במערת המכפלה .  הארבעה הביעו את דעתם ואמרו כי האתר חייב להיכלל במסגרת רשימת אתרי המורשת הלאומיים .
